# CyberSecurity Awareness Bot

## Overview
This is a console-based chatbot built using C# and .NET.  
It helps users learn about cybersecurity topics such as phishing, malware, passwords, and safe browsing.

## Features
- Voice greeting using audio playback
- Interactive chatbot
- Typing animation for responses
- Cybersecurity awareness tips
- Colored console UI

## Technologies Used
- C#
- .NET 10
- Visual Studio
- GitHub Actions (CI)

## How to Run
1. Open the project in Visual Studio
2. Build and run the application
3. Enter your name
4. Ask cybersecurity-related questions
